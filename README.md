## **Features**

- CRUD actions
- JSON Web Token Authentication
- CORS to only allow requests from my frontend
- RESTful API

## **Outcome**

- Used **Express**
- Used **MongoDb Atlas**
- Used **express-validator**
- Used **luxon**
- Used **debug**
- Used **dotenv**
- Used **bcrypt**
- Used **passport-jwt**
- Used **jwt**
- Used **cors**

## **Getting Started**

HTTPS

```bash
git clone https://github.com/minhhoccode111/blog-back.git
```

or SSH

```bash
git clone git@github.com:minhhoccode111/blog-back.git
```

then

```bash
cd blog-back
npm install
npm start
```
